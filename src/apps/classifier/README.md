# OxydHarmonics App

## Requirements:
* [Docker](https://docs.docker.com/get-docker/)
* [Docker compose plugin](https://docs.docker.com/compose/install/linux/)
* make

## Installation

```sh
git clone git@github.com:wpradier/OxydMelodics.git
cd OxydMelodics/src/apps/classifier
make up
```

## Check dependencies
```sh
make deps
```
